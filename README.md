## 🧩 1. Sliding Puzzle — اللغز المنزلق

**لعبة اللغز المنزلق الكلاسيكية.** الهدف: رتّب القطع بالترتيب الصحيح بأقل عدد من الحركات وأسرع وقت.

### 🎮 أوضاع اللعب

| الوضع | الوصف |
|-------|-------|
| 🔢 **أرقام** | رتّب الأرقام من 1 للأمام بالترتيب الصحيح |
| 🔤 **حروف** | رتّب الحروف الإنجليزية بالترتيب الأبجدي |
| 🖼️ **صور** | ارفع صورة مخصصة وشاهدها تتقطع — رتّبها مجدداً |

### ⚙️ مستويات الصعوبة

| المستوى | الشبكة | عدد القطع |
|---------|--------|-----------|
| 🟢 سهل | 3 × 3 | 8 قطع |
| 🟡 متوسط | 4 × 4 | 15 قطعة |
| 🔴 صعب | 5 × 5 | 24 قطعة |

### 🤖 الحل التلقائي — A\* Algorithm

عند الضغط على **🤖 حل**، تبدأ خوارزمية **A\*** في إيجاد أقصر مسار لحل اللغز:
- **Heuristic**: Manhattan Distance
- تحريك القطع تلقائياً بتأخير 420ms لرؤية كل خطوة
- شريط يُظهر عدد الخطوات المتبقية
- إمكانية إيقاف الحل في أي لحظة

### 👻 Ghost Mode

تظهر أرقام خفيفة في زاوية كل خلية تُرشدك إلى القيمة الصحيحة. القطع الصحيحة تتوهج بالأخضر.

### 🔊 المؤثرات الصوتية

أصوات تفاعلية مولَّدة بـ **Web Audio API** بدون ملفات خارجية.

## ♟️ 2. الداما المغربية — Moroccan Checkers

**لعبة الداما المغربية التقليدية** ضد ذكاء اصطناعي قوي — أو ضد صديق.

### 🤖 نظام الذكاء الاصطناعي

| الميزة | القيمة |
|--------|--------|
| **الخوارزمية** | Minimax + Alpha-Beta Pruning |
| **عمق البحث** | حتى **8** (يتكيف مع عدد القطع: 50+ قطعة ← عمق 6، أقل من 10 قطع ← عمق 8) |
| **ترتيب الحركات** | Smart Move Ordering لزيادة كفاءة Alpha-Beta |
| **المعالجة** | **Web Worker** حقيقي — لا يجمّد الواجهة أثناء التفكير |
| **التقييم** | `evalBoard()` متعدد العوامل (القطع، الملكات، التهديدات، السيطرة) |

### 🎯 القوانين المطبّقة

- ✅ **الجندي**: يتحرك للأمام قطرياً فقط (اتجاهين)
- ✅ **الجندي**: يأكل للأمام قطرياً فقط (اتجاهين)
- ✅ **الملكة**: تطير وتأكل في الاتجاهات الأربعة
- ✅ **لا ترقية أثناء سلسلة الأكل**
- ✅ **الأكل الإجباري** — إذا تقدر تاكل، لازم تاكل
- ✅ **سلسلة الأكل** — يأكل القطع تباعاً

### 🎮 أنماط اللعب

- 🧑 **ضد الكمبيوتر** — العب ضد AI (أحمر ضد أزرق)
- 👥 **لاعبان** — العب مع صديق على نفس الجهاز

## 🎮 3. ولد الحارة — Mario Platformer

**لعبة منصات كاملة** بروح ماريو الكلاسيكي، بقلب عربي — مع اسم مستعار "ولد الحارة".

### 🏗️ محرك اللعبة

| المكون | التفاصيل |
|--------|----------|
| **المحرك** | Canvas 2D — كامل من الصفر |
| **عامل الجاذبية** | 0.55 — فيزياء مخصصة |
| **الشخصية** | حركة ← → ↑ ↓ + قفز مزدوج + قرفصاء |
| **حجم الخلية** | 36px — نظام Grid للـ Tile-based levels |
| **الرسوم** | ألوان صلبة + تدرجات — بدون صور خارجية |

### 🗺️ المستويات

**5 مستويات** كاملة، كل مستوى بتصميم فريد:

| المستوى | الوصف |
|---------|-------|
| 🟢 **Level 1** | البداية — أرض صلبة + عملات + أعداء بسيطون |
| 🟡 **Level 2** | منصات متحركة + عوائق |
| 🔵 **Level 3** | كهف — إضاءة منخفضة + أعداء في الظلام |
| 🟠 **Level 4** | سماء — منصات عائمة + قفزات دقيقة |
| 🔴 **Level 5** | المرحلة النهائية — Boss + تحديات قصوى |

### ⭐ العناصر

| العنصر | الوظيفة |
|--------|---------|
| 🪙 **عملات** | تجميعها يزيد النقاط — تطفو لأعلى/أسفل (bobbing) |
| 🦴 **أعداء** | أنواع متعددة — تمشي، تطير، تتبع |
| ❤️ **قلوب** | 3 أرواح — تخسر روح عند لمس عدو |
| 🟩 **بيض مفاجآت (Power-ups)** | 🏃 سرعة، 💪 قوة، 🛡️ درع |
| 🧊 **كتل قابلة للكسر** | تضربها من تحت تطلع عملة أو مفاجأة |
| 🚩 **علم النهاية** | توصل للعلم تربح المستوى |

### 🎵 المؤثرات

- موسيقى باكجرواند (Web Audio API)
- 🪙 صوت عند جمع عملة
- 💥 صوت عند كسر عدو
- 🏁 موسيقى نصر عند إكمال المستوى

## 🃏 4. Memory Game — لعبة الذاكرة

**لعبة الذاكرة الكلاسيكية** بالايموجي — اختبر ذاكرتك!

### 🎮 طريقة اللعب

- بطاقات بإيموجي متنوعة — اقلب 2 متطابقين عشان تمسكهم
- كل الأيموجي تتخبأ — ركّز وتذكر مواقعهم

### 📊 مستويات

- **4 × 4** — 8 أزواج (سهل)
- **6 × 6** — 18 زوجاً (صعب)

### 🎯 الميزات

- ✅ رسوم متحركة عند القلب والتطابق
- ✅ عداد المحاولات
- ✅ مؤقت زمني
- ✅ متصدر نتائج محلي
- ✅ تصميم أنيق مع توهجات

## 🐍 5. Snake — التعبان

**لعبة الثعبان الكلاسيكية** بأسلوب **CRT أخضر رجعي** — كأنك تلعب على شاشة قديمة!

### 🖥️ الأسلوب البصري

- خلفية سوداء كالأنبوب القديم
- الثعبان والطعام بلون **أخضر فسفوري**
- خطوط مسح CRT (scanlines)
- تأثير اهتزاز خفيف للشاشة (retro feel)

### ⚡ مستويات السرعة

| المستوى | السرعة (ms) |
|---------|-------------|
| 🐢 بطيء | 200ms — للمبتدئين |
| 🐇 متوسط | 140ms — التحدي |
| 🚀 سريع | 75ms — للأساطير |

### 🎮 التحكم

| الجهاز | الطريقة |
|--------|---------|
| 🖥️ **لوحة المفاتيح** | ↑ ↓ ← → أو WASD |
| 📱 **اللمس** | swipe في أي اتجاه (Touch Events) |
| ⏸️ **Pause** | زر Pause — أو المس/اضغط Space |

### 🏆 النتائج

- كل التفاحات اللي تاكلها تزيد طولك ونقاطك
- Score يُعرض مباشرة
- إذا حكيت الحائط أو جسمك — Game Over

## 📱 Progressive Web App

- **Service Worker v3** — Cache-First، حذف تلقائي للـ cache القديم
- **Web App Manifest** — دعم RTL وعربي كامل
- **Install Banner** — تثبيت مباشر من المتصفح
- **يعمل بالكامل Offline** بعد أول زيارة
- **جميع الألعاب الخمس** متاحة بدون إنترنت
